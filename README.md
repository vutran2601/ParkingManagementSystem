# Parking Management System

**MAIN FEATURES:**
 1. Vehicle check-in
 2. Vehicle check-out
 3. Fuel charge *(Gasoline or Diesel)*
 4. Change oil
 5. Washing

*Just follow these steps below:*

**Run NodeJS**
 - cd server
 - npm i
 - npm start

**Run ReactJS**
 - cd client
 - npm i
 - npm start
